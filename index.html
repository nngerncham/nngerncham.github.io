About me

Hi! My name is Nawat Ngerncham. I am a 4th-year undergraduate student at Mahidol University International College with a major in Computer Science and a minor in Applied Mathematics.

I am interested in graph algorithms, parallel algorithms, and the concept of randomness in mathematics.   • EigenSeeker: a Home-baked Eigenvalue/vector Finder

    2021

    A program to numerically find the eigenvalue and eigenvector of a given matrix using the power method, the shifted inverse power method, and the Gershgorin circle theorem.

    The GitHub repository can be found here.

  • NumPar: A NumPy Clone with Parallelism Baked In

    2022

    A Python library for numerical computation based on the library NumPy, specifically its linear algebra module (numpy.linalg). The library is implemented is Rust, using Rayon for parallelism and PyO3 for binding Rust to Python.

    The GitHub repository can be found here.
